# 这里是像素奥德赛的`OldVersion`分支空间
创立此仓库的目的是为了方便团队协作，减轻coder的压力
## 一：我修改了什么？（每次发布都会更新）
+ 核对了作物出售价格与时间（%1）  
+ 替换了上层背景，取消了装饰图层（%2）
## 二：怎么使用？
+ 上一个版本的源码请在OldVersion分支下载  
+ 而最新版请在main分支下载  
> 新版更新好后，请把代码上传到main分支  
代码备份放在OldVersion分支
## 三：有什么建议？
